# 마이링크 (My Link) 사용자 시나리오 (User Scenarios)

## 1. 방문자: 마이링크 페이지 방문 과정

* **프로필 접속**: 방문자는 **크리에이터가 등록한 여러 외부 채널을 한곳에서 모아보기 위해서** 방문자용 고유 웹 프로필 페이지(`/displayName`)에 **접속한다**.
* **프로필 열람**: 방문자는 **크리에이터의 아이덴티티와 간단한 정보를 파악하기 위해서** 상단에 노출된 프로필 이미지, 닉네임, 한 줄 소개(Bio)를 **열람한다**.
* **링크 클릭**: 방문자는 **특정 관심 채널이나 목적지로 이동하기 위해서** 리스트에 나열된 (파비콘 아이콘이 포함된) 링크 블록 버튼을 **클릭한다**.
* **새 창 전환**: 방문자는 **마이링크 원본 프로필 화면의 탐색 흐름을 방해받지 않고 목적지 사이트를 확인하기 위해서** 새로 열리는 탭(`_blank`)에서 외부 사이트를 **탐색한다**.

## 2. 소유자: 링크 추가/수정/삭제 과정

* **링크 추가 진입**: 소유자는 **자신의 새로운 소셜 미디어나 포트폴리오 채널을 방문자에게 노출하기 위해서** 관리자 화면(대시보드)에서 '링크 추가' 버튼을 **클릭한다**.
* **링크 등록**: 소유자는 **방문자에게 이동할 링크의 목적을 명확히 설명하기 위해서** 목적지 URL과 링크 제목(Title)을 직접 타이핑하여 **입력한다**. *(이때, 구글 파비콘 API를 통해 기본 아이콘이 자동 할당된다.)*
* **인라인 내용 수정**: 소유자는 **이미 등록된 기존 링크의 제목 오타나 URL을 쉽고 빠르게 변경하기 위해서** 별도의 모달 창이나 페이지 이동 없이 해당 항목의 텍스트 영역을 직접 클릭해 인라인(Inline Edit)으로 **수정한다**.
* **노출 상태 전환**: 소유자는 **이벤트 종료 등의 이유로 방문자에게 링크를 잠시 가려두되, 관련 기록은 버리지 않고 나중에 재사용하기 위해서** 노출 활성화 토글(ON/OFF)을 클릭하여 **숨김(OFF) 처리한다**.
* **링크 삭제**: 소유자는 **관리 리스트에서 완전히 불필요해진 링크 데이터를 영구적으로 지우기 위해서** 해당 링크 항목의 우측에 위치한 삭제 아이콘을 **클릭한다**.
* **삭제 안전 재확인**: 소유자는 **단순 클릭 실수로 인해 애써 기입한 링크 정보가 돌이킬 수 없이 날아가는 것을 방지하기 위해서** 나타나는 "정말 삭제하시겠습니까?" 팝업창에서 재차 '확인'을 **누른다**.
